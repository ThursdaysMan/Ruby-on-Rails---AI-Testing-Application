# Sirius - Contrastive AI-Testing Satellite Imagery Application

A test application using AI modelling - Sirius uses contrastive AI modelling to classify land use for multiple use cases, a world model for anomaly detection and for new class creation/classification and predictive modelling of events.


Main MVC functionality using Ruby on Rails

Gems:
- Rails
- TensorFlow
- Numpy
- Chroma-DB
